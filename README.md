# The Landing Pad

The goal of this project is to produce a [“No Rights Reserved”](https://creativecommons.org/public-domain/cc0/) 3D model
of the [world's first UFO Landing Pad](https://www.stpaul.ca/visitors/ufo-landing-pad), built in 1967 in the town of [St. Paul, Alberta, Canada](https://en.wikipedia.org/wiki/St._Paul,_Alberta)

> This repository will be made public after a successful print is made on the 3D printer at the [St. Paul Public Library](https://www.stpaul.ca/residents/public-library).

# 3D Printer

- Model: [Prusa i3 Mk3s](https://www.makerhacks.com/prusa-mk3s-review/)
- Dimentions: 250mm x 210mm x 200mm

# LiDAR scan: 2025-06-13.stl

Using a free trial of [poly.cam](https://poly.cam/) on iPhone I made a LiDAR scan. Seems this is a good way to capture the rocks under the landing pad. The rest of the structure should probably be modeled from scratch using measurements from the scan.
